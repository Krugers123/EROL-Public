# EROL What It Is / Is Not

## Purpose

This document clarifies the public-safe identity of EROL.

## What EROL Is

EROL is:

- an epidemic response observability layer
- a proportionality-focused monitoring framework
- a trajectory and calibration analysis surface
- a neutral external interpretation layer

## What EROL Is Not

EROL is not:

- a medical authority
- a treatment system
- a political enforcement tool
- a system for controlling populations
- a substitute for health governance

## Why This Distinction Matters

Large-scale crisis systems become easier to misread when:

- observability is confused with authority
- calibration is confused with ideology
- signal analysis is confused with enforcement

EROL is strongest when it remains:

- technical
- neutral
- external
- proportionality-focused

## Public Reading

The best public reading of EROL is:

- external observability for epidemic response systems
- drift and calibration analysis over time
- proportionality monitoring across multiple signal families
